# AlgoWiki 文档索引

本文档目录用于区分三类信息：

- 面向项目负责人和开发者的总体说明：[项目说明书](./项目说明书.md)
- 面向最终使用者和后台操作人员的功能说明：[详细使用手册](./详细使用手册.md)
- 面向运维和长期维护的部署、巡检、备份和故障处理说明：[维护手册](./维护手册.md)
- 面向远程服务器、Docker、镜像上传和正式切换的实操说明：[远程部署指南](./远程部署指南.md)
- 面向公开发布前检查的说明：[GitHub发布清单](./GitHub发布清单.md)

建议阅读顺序：

1. 首次接手项目：先看《项目说明书》
2. 准备培训使用者：看《详细使用手册》
3. 准备上线、迁移或值守：先看《维护手册》，再按《远程部署指南》执行

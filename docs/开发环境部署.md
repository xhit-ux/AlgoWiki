# 基于 Linux 的 AlgoWiKi 开发环境部署

>本教程基于Ubuntu-24.04编写，默认以 `root` 权限运行命令</br>
>开发模式具有热重载，可以对前段实时编译，所以**请确保您的内存至少有4G的可用空间**</br>

## 一、MySQL安装

安装MySQL
```
apt update
apt install mysql-server
```

使用`mysql -u root`登录默认账号

由于后续需要使用数据库的密码，所以需要为默认账户创建一个密码。
```
ALTER USER 'root'@'localhost' IDENTIFIED WITH caching_sha2_password BY '你要设置的新密码';
FLUSH PRIVILEGES;
```
改完以后退出
```
exit
```
使用账密登录测试：
```
mysql -u root -p
```
这时输入你刚才设置的密码。注意：改成密码认证后，`sudo mysql -u root` 将无法使用，因为 `root` 已经不再走 `auth_socket`。这个变化是由认证插件切换导致的。

## 二、venv安装

安装venv
```
apt update
apt install python3-venv
```
安装完成后进入root目录创建虚拟环境
```
cd ~
python3 -m venv /root/.venvs/algowiki
source /root/.venvs/algowiki/bin/activate
```
确认安装完成即可退出虚拟环境
```
deactivate
```

## 三、安装AlgoWiKi

将AlgoWiKi安装到`root`目录下
```
git clone https://github.com/NullResot/AlgoWiki.git /root/AlgoWiki --recursive
cd /root/AlgoWiki
```
配置环境变量，复制 `backend/.env.example` 为 `backend/.env`，至少配置：
```
DB_ENGINE=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_NAME=algowiki
DB_USER=root
DB_PASSWORD=your-password
```
然后进入项目激活虚拟环境
```
cd /root/AlgoWiki
source /root/.venvs/algowiki/bin/activate
```
然后创建MySQL 数据库
```
mysql -h 127.0.0.1 -P 3306 -u root -p -e "CREATE DATABASE IF NOT EXISTS algowiki CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;"
```
安装依赖
```
/root/.venvs/algowiki/bin/python -m pip install --upgrade pip
/root/.venvs/algowiki/bin/python -m pip install -r backend/requirements.txt
```
导入数据库
```
/root/.venvs/algowiki/bin/python backend/manage.py migrate --noinput
/root/.venvs/algowiki/bin/python backend/manage.py seed_initial_data \
  --content-file backend/data/xcpc_readme_snapshot.md \
  --reset-superadmin-password \
  --superadmin-password 'AlgoWiki@2026'
```

## 四、启动后端

```
cd /root/AlgoWiki
source /root/.venvs/algowiki/bin/activate
/root/.venvs/algowiki/bin/python backend/manage.py runserver 127.0.0.1:8001
```
检测是否正常工作
```
curl -fsS http://127.0.0.1:8001/api/health/
```

## 五、启用前端

安装Node.js20
```
apt-get update
apt-get install -y curl ca-certificates gnupg
curl -fsSL https://deb.nodesource.com/setup_20.x | bash -
apt-get install -y nodejs
```
然后启动前端
```
cd /root/AlgoWiki/frontend
npm ci
npm run dev
```
启动后默认访问`http://127.0.0.1:5173`

 - 密码：AlgoWiki@2026
